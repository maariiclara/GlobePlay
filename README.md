# 🌍 GlobePlay

Projeto acadêmico desenvolvido em grupo durante o curso técnico no **SENAI**, com o objetivo de criar uma plataforma inspirada em serviços de streaming, focando na organização visual de conteúdos e na construção de interfaces web.

---

## ✨ Sobre o Projeto

O site foi desenvolvido com uma proposta semelhante a plataformas de streaming, contendo páginas dedicadas a séries, organização de conteúdos e navegação entre seções.

O projeto teve como foco a prática de desenvolvimento front-end, trabalhando estruturação de páginas, estilização e componentes visuais.

---

## 🤝 Projeto em Grupo

Este projeto foi desenvolvido em equipe durante atividades acadêmicas no SENAI.
